# Prompt Archive — Mini-Project -- Focus Sound Player

All AI prompts used for this mini-project must be saved here, both for electronic submission and inclusion in the group Project Manual.

## Naming convention

Use dated, descriptive filenames:

```
YYYY-MM-DD_short-description.md
```

Example: `2026-05-19_initial-architecture-and-fastapi-split.md`

## What to archive

- Prompts used to **design** the mini-project
- Prompts used to **write or refactor** implementation code
- Prompts used to **write documentation** (README, Project Manual sections)

## Template for each archived prompt

Copy `TEMPLATE.md` for each new entry and fill in the sections.
